About Me

Nothing drives me more than the pursuit of knowledge. As I age, I find my desire to learn only grows stronger. About a year age, I decided to let my thirst for knowledge lead me where it would. I soon found myself immersed in the world of coding. The more I studied, the more I recognized the perfect path coding is for someone so driven by learning because, as I am discovering each day, there is always something new just around the corner. Projects

Conference Demo Site

A landing page for a fictional conference that uses Bootstrap for the layout, design, and functionality.

Conference Demo Site

Parameters:

Create a site based on a client mockup using the Bootstrap framework.

Challenges:

Bootstrap was a new framework for me. I had trouble getting my head around the grid system in the beginning. Bootstrap can be very limiting, so adjusting small details or style choices felt almost impossible.

Takeaways:

Bootstrap is a tool that I will make use of in certain situations. Bootstrap could save a lot of time on projects that don't require a lot of intricate customization. Personally, I still enjoy writing my own Sass.
